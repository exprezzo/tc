In this sample, the wijgrid widget shows the same data as in the Original Data grid, but the column data types in the wijgrid are explicitly specified:

  • ProductID: dataType = "string"
  • UnitPrice: dataType = "currency"
  • Quantity: dataType = "number"
  • Discount: dataType = "number", dataFormatString = "p0"
  • Discontinued: dataType = "boolean"

Try to sort the UnitPrice column in both grids to see a difference.
